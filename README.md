### Forum

**This project is a platform for communication within the job4j community.**

In the forum you can discuss topics in a project with another users:
* post questions
* comment on it

#### Dependencies

* Spring Boot 2 
* Spring Security
* Spring Data JPA with Hibernate + PostgreSQL
* Java 11
* JSP(JSTL)
* Junit, Mockito, H2
* Maven
* Log4j
* Liquibase
